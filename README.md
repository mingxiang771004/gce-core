# 几何化认知引擎(GCE) v1.0

全球首个基于复杂系统演化公理的低幻觉、强因果通用认知引擎。基于开源大模型增量改造，无需从头训练，事实性错误率<5%，长程因果准确率>90%。

## 项目负责人声明

本套几何化认知引擎完整理论框架由本人独立提出，全程负责项目学术框架搭建、核心约束规则制定与科研层面背书。软件代码工程化实现、迭代优化等落地工作开放给全球开源开发者共同协作完成。

## 贡献说明

所有代码实现必须严格遵循官方文档第3章“核心算法规则”，任何违反核心规则的PR将不会被合并。

欢迎提交PR参与MVP开发、文档优化、测试验证等工作。

MVP v0.1正式发布时，所有核心贡献者将被列入官方开发者名单。

关于核心理论与规则的问题，请通过邮箱联系我；工程实现问题请在Issue区讨论。

## 提交PR须知

1.请附带文字说明，阐述代码实现如何契合项目核心算法规则；

2.功能代码需同步提交单元测试，标注核心规则模块测试覆盖率；

3.无说明、无测试的PR将直接关闭，不予审核。
